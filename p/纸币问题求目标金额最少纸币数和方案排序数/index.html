P2842 纸币问题 1&2

题目描述

某国有 $n$ 种纸币，每种纸币面额为 $a_i$ 并且有无限张，现在要凑出 $w$ 的金额，试问最少用多少张纸币可以凑出来？（保证可以凑出对应金额）

输入格式

第一行两个整数 $n,w$，分别表示纸币的种数和要凑出的金额。
第二行一行 $n$ 个以空格隔开的整数 $a_1, a_2, a_3, \dots a_n$ 依次表示这 $n$ 种纸币的面额。

输出格式

一行一个整数，表示最少使用的纸币张数。

输入输出样例 #1

输入 #1 

1
2

6 15
1 5 10 20 50 100

输出 #1 

1

2

输入输出样例 #2

输入 #2 

1
2

3 15
1 5 11

输出 #2 

1

3

说明/提示

对于 $40%$ 的数据，满足 $n\le 10$，$w\le 100$；
对于 $100%$ 的数据，满足 $1\le n\le 10^3$，$1\le a_i , w\le 10^4$。

最少纸币数的解决

思路

dp数组除了下标0统一初始化为一个很大的数，dp[0]初始化为0,因为0元需要0张纸币这样当有更小的数的可能性的时候，就可以更新纸币数。 外层是纸币，内层是金额。 如果把dp[j]盒子中价值vals[i]的部分替换为一张价值为vals[i]的纸币,也就是让dp[j]变成dp[j - vals[i]]+1,如果这样可以让dp[j]变得更小,也就是纸币数更少,那么就替换掉.

    std::vector<int> dp(w + 1, 1e9);
    dp[0] = 0;
    for (int i = 0; i < n; ++i)
    {
        for (int j = vals[i]; j <= w; j++)
        {
            if (dp[j - vals[i]] + 1 < dp[j])
            {
                dp[j] = dp[j - vals[i]] + 1;
            }
        }
    }

P2840 纸币问题 2

题目背景

你是一个非常有钱的小朋友。

题目描述

你有 $n$ 种面额互不相同的纸币，第 $i$ 种纸币的面额为 $a_i$ 并且有无限张，现在你需要支付 $w$ 的金额，求问有多少种方式可以支付面额 $w$，答案对 $10^9+7$ 取模。
注意在这里，同样的纸币组合如果支付顺序不同，会被视作不同的方式。例如支付 $3$ 元，使用一张面值 $1$ 的纸币和一张面值 $2$ 的纸币会产生两种方式（$1+2$ 和 $2+1$）。

输入格式

第一行两个正整数 $n,w$，分别表示纸币的种数和要凑出的金额。
第二行一行 $n$ 个以空格隔开的正整数 $a_1, a_2, \dots a_n $ 依次表示这 $n$ 种纸币的面额。

输出格式

一行一个整数，表示支付方式的数量。

输入输出样例 #1

输入 #1 

1
2

6 15
1 5 10 20 50 100

输出 #1 

1

42

输入输出样例 #2

输入 #2 

1
2

3 15
1 5 11

输出 #2 

1

39

说明/提示

对于 $40%$ 的数据，满足 $n\le 10$，$w\le 100$；
对于 $100%$ 的数据，满足 $1\le n\le 10^3$，$1\le a_i , w\le 10^4$。

其实小朋友并没有那么多钱。

排序数的解决

思路

此时dp盒子中的内容不是纸币数而是方案数.个人认为dp盒子初始化为什么内容取决于要求什么,比如上文要求最小值那么就需要初始化为大数然后替换得到小数,如果是求方案数那么就需要从0开始积累.

所以dp盒子除了0初始化为1,其他都初始化为0.(凑出0元有1种方案)

由于是排序数,所以你可以塞入很多张一样的纸币.对于循环,外层是金额,内层是纸币.如果当前i大于vals[j]那么就可以用vals[j]来凑,让vals[j]作为最后一张纸币塞入盒子,然后和前面i减去vals[j]对应的盒子的方案累加.

    std::vector<int> dp(w + 1, 0);
    const int mod = 1e9 + 7;
    dp[0] = 1;

    for (int i = 1; i <= w; i++)
    {
        for (int j = 0; j < n; j++)
        {
            if (i >= vals[j])
            {
                dp[i] = (dp[i] + dp[i - vals[j]]) % mod;
                // 如果我们要放入这个val，那么就需要提前凑i - val元，所以需要累加i - val的dp
            }
        }
    }
